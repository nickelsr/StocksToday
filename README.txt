Unix:
In Terminal, navigate to folder containing project files
install dependencies: python3 -m pip install -r requirements.txt
run program: python3 stocks_today.py

Windows:
In Command Prompt, navigate to folder containing project files
install dependencies: py -m pip install -r requirements.txt
run program: py stocks_today.py

Using program:
Enter a string to display query results, and find out if the tick symbol provided is valid.
Valid symbols vary from one to several characters in length.
Examples of valid symbols include:
V, for Visa Inc.,
ISRG, for Intuitive Surgical Inc.,
GOOG, for Alphabet Inc.,
etc.
